# Scoring Criteria

## Bloom's Taxonomy Assessment Framework

### Cognitive Levels (1-6)
1. **Remember** (基础记忆): Recall facts, terms, concepts
2. **Understand** (理解): Explain ideas, summarize, interpret
3. **Apply** (应用): Use knowledge in new situations, execute procedures
4. **Analyze** (分析): Break down information, identify patterns, distinguish components
5. **Evaluate** (评估): Justify decisions, critique, assess validity
6. **Create** (创造): Design, construct, produce original work

## Partial Credit Rubric

### Score Distribution
- **Full Credit (100%)**: Demonstrates target cognitive level with accuracy
- **Strong (80-90%)**: Minor gaps in execution, core understanding solid
- **Developing (60-75%)**: Partial mastery, key concepts present but incomplete
- **Emerging (40-55%)**: Basic attempt, significant gaps, shows effort
- **Minimal (20-35%)**: Fragmented understanding, needs substantial support
- **No Credit (0-15%)**: Off-topic, no evidence of engagement

### Partial Credit Guidelines
- Award points for correct reasoning even if final answer is wrong
- Recognize valid alternative approaches
- Credit intermediate steps in multi-step problems
- Value self-correction and metacognitive awareness

## Improvement Tracking

### Progress Indicators
- **Trajectory**: Compare current performance to baseline (first 3 attempts)
- **Consistency**: Track stability across similar difficulty levels
- **Velocity**: Rate of improvement over time windows (weekly/monthly)
- **Depth**: Movement up Bloom's taxonomy levels

### Tracking Metrics
```
improvement_score = (current_avg - baseline_avg) / baseline_avg * 100
consistency_score = 1 - (std_dev / mean) * 100
bloom_progression = current_level - initial_level
```

## Process vs Outcome Balance

### Process Evaluation (40%)
- Problem-solving approach and strategy selection
- Use of resources (notes, references, prior knowledge)
- Self-monitoring and error detection
- Persistence through difficulty
- Quality of questions asked

### Outcome Evaluation (60%)
- Correctness of final answer
- Completeness of solution
- Accuracy of understanding
- Application of concepts
- Demonstration of mastery

## Effort Recognition

### Effort Indicators
- Time invested relative to task complexity
- Number of revision attempts
- Depth of exploration beyond minimum requirements
- Engagement with feedback
- Voluntary practice outside assigned work

### Effort Scoring
- **High Effort + High Achievement**: Full recognition, exemplar status
- **High Effort + Low Achievement**: Growth mindset praise, targeted support
- **Low Effort + High Achievement**: Challenge with harder material
- **Low Effort + Low Achievement**: Motivational intervention needed

## Scoring Formula

```
final_score = (
  bloom_level_score * 0.30 +
  accuracy_score * 0.30 +
  process_score * 0.20 +
  improvement_delta * 0.10 +
  effort_score * 0.10
)
```

## Feedback Principles

- Specific: Reference exact work, not generalities
- Actionable: Provide clear next steps
- Timely: Deliver while context is fresh
- Balanced: Acknowledge strengths and growth areas
- Growth-oriented: Frame challenges as opportunities

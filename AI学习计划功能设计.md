# AI学习计划功能设计

## 此功能旨在创建一整套可持续的，完整的学习计划表

- 用户输入
  - 当前计划的最终目标
  - 要为之付出的时间
  - 开始-结束日期 开始日期支持自动获取
  - 当前状态
- AI负责
  - 理解用户的输入
  - 询问缺失信息并接受补充
  - 结合用户的个人信息
  - 输出一个完整的计划表 总结为年→月→天→单计划（根据复杂度由ai决定生成到哪一层）
  - 此计划表要可以由创建不同主题
    - 举例：
      ``` text
      {
        创建时间
        计划时间段
        主题1(e. 数学)
        {
            预计时间
            次级主题1(e. 函数)
            {
                预计时间
                月计划
                {
                    预计时间
                    周计划
                    {
                        预计时间
                        天计划
                        {
                            预计时间
                            详细计划
                        }
                    }
                }
            }
        }
        主题2(e. 语文)
        {
            预计时间
            次级主题1
            {
                预计时间
                月计划
                {
                    预计时间
                    周计划
                    {
                        预计时间
                        天计划
                        {
                            预计时间
                            详细计划
                        }
                    }
                }
            }
        }
      }
      ```
    - 创建的计划要求可以被推迟 提前完成 如果选择推迟可由用户手动触发ai优化日程
